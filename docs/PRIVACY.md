# Privacy Policy

## Data collection

The app does not collect, transmit, sell, or share personal data or usage data.
It does not include analytics, advertising, tracking, account, payment, or
server SDKs.

## Images and project documents

Images selected from Photos, Files, or the camera are processed locally on the
device. Project documents and exported PNG/CSV files remain in locations chosen
by the user. The developer cannot access or recover these files.

## Permissions

Camera access is requested only when the user selects the camera import action.
Photo selection uses Apple's system Photos picker and does not request broad
photo-library access.

## Deletion

Users can delete project and export files through the Files or Photos apps and
can remove the application at any time.
